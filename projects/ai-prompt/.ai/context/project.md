# 项目概述

## 项目名称
AI Prompt Manager - 提示词管理工具

## 项目定位
一个纯前端提示词管理网站，帮助用户管理、组织和复用AI提示词模板。

## 技术栈
- **前端框架**: React 18 + TypeScript 5.9
- **构建工具**: Vite 7.3
- **状态管理**: Zustand 5.0
- **UI框架**: Ant Design 5.x (最新稳定版)
- **样式方案**: Ant Design 主题 + 自定义CSS
- **数据持久化**: IndexDB (原生API)
- **包管理器**: npm

## 核心功能
1. **提示词模板** - 创建、编辑、删除、收藏、搜索提示词
2. **全局变量** - 定义可复用的变量值
3. **标签管理** - 为模板添加标签进行分类
4. **历史记录** - 自动记录每次使用模板的执行历史
5. **数据导入导出** - JSON格式备份和迁移数据

## 项目状态
- 已完成初版开发
- 所有功能已实现并测试通过
- UI已升级为Ant Design

## 目标用户
- AI工具使用者（ChatGPT、Claude、Midjourney等）
- 提示词工程师
- 内容创作者
